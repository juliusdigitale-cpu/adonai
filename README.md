# Adonai FotoCine 📸🎬

Bem-vindo ao **Adonai FotoCine** — um website moderno e elegante para fotografia e filmagem profissionais.

## 🎯 Sobre o Projeto

Adonai FotoCine é um site responsivo e sofisticado criado para apresentar serviços de fotografia e filmagem profissionais. O projeto combina design elegante com funcionalidade prática, oferecendo uma excelente experiência ao usuário.

**Slogan:** _Capturando emoções com arte_

## ✨ Principais Características

- 🎨 **Design Elegante e Moderno** - Interface limpa com tema sofisticado
- 📱 **Totalmente Responsivo** - Funciona perfeitamente em dispositivos móveis, tablets e desktops
- 🎭 **Seções Organizadas** - Hero, serviços, portfólio e contato bem estruturados
- 📧 **Formulário de Contato Funcional** - Integração com mailto para agendamentos
- ⚡ **Performance Otimizada** - Lazy loading de imagens e código CSS minificado
- 🎪 **Portfólio Visual** - Galeria de trabalhos e projetos anteriores

## 🚀 Como Usar

### Pré-requisitos
- Um navegador web moderno (Chrome, Firefox, Safari, Edge)
- Nenhuma dependência externa ou compilação necessária

### Instalação Local

1. **Clone o repositório:**
```bash
git clone https://github.com/juliusdigitale-cpu/adonai.git
cd adonai
```

2. **Abra o arquivo no navegador:**
- Duplo clique em `index.html`, ou
- Use um servidor local (recomendado):
```bash
# Com Python 3
python -m http.server 8000

# Com Node.js (http-server)
npx http-server
```

3. **Acesse em seu navegador:**
```
http://localhost:8000
```

## 📱 Responsividade

O site é otimizado para:
- **Desktops** (1100px+) - Layout completo com 2 colunas
- **Tablets** (até 860px) - Layout adaptado com 1 coluna
- **Smartphones** (até 560px) - Navegação e espaçamento ajustados

## 🔧 Funcionalidades

### Formulário de Contato
O formulário realiza:
- ✅ Validação de campos obrigatórios
- ✅ Verificação de formato de e-mail
- ✅ Feedback visual de sucesso/erro
- ✅ Abertura do cliente de e-mail com pré-preenchimento

### Performance
- 📦 Lazy loading de imagens no portfólio
- 🎯 Eager loading da imagem hero
- ⚡ CSS otimizado e minificado
- 🔗 Smooth scroll navigation
